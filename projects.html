HaM3D: XR Framework for Haptic Human-Robot Interaction

Goal: To enhance remote Human-Robot Interaction (HRI) for complex industrial tasks (like assembly and welding) by developing an intuitive, multimodal Extended Reality (XR) framework with integrated haptic feedback.

Developed HaM3D (Haptic and Multimodal framework for 3D Display), an immersive XR (VR/MR) interface addressing limitations in traditional teleoperation. The system integrates visual (virtual replicas, video), auditory (speech, sound), text, and haptic feedback. Haptic guidance using the Artificial Potential Field (APF) method assists operators in collision avoidance. Computer vision models interpret the remote scene for presentation in XR. Validated through user studies involving industrial tasks like peg-in-hole assembly and arc welding.

Key Technologies: Extended Reality (XR) Unity (VR/MR) HoloLens 2 HTC Vive Haptic Feedback (APF) Human-Robot Interaction (HRI) Computer Vision Robot Teleoperation C# Python

Optimized Factory Layout Planning in the Industrial Metaverse

Goal: To improve factory layout planning efficiency and decision-making by combining Genetic Algorithm optimization with immersive visualization in an Industrial Metaverse.

Developed an approach using a Genetic Algorithm (GA) to optimize the relative placement of machines and assembly lines within a defined factory floor space. The algorithm considers constraints like machine proximity, required production rates, material flow, and available area. The optimized layout solutions are then rendered and visualized within an interactive Industrial Metaverse environment built using Unity, allowing users to experience and evaluate the proposed layouts immersively via VR headsets before physical implementation. This method facilitates better spatial understanding, incorporates safety considerations (e.g., visual safety lines), and aims to improve space utilization and production efficiency compared to traditional methods.

Key Technologies: Genetic Algorithms (GA) Factory Layout Optimization Industrial Metaverse Virtual Reality (VR) Unity Process Visualization C# Python Web Sockets

LinguaLink: In-House AI Translation Tool

Goal: To address high costs, vendor dependency, and delays in technical document translation by developing an efficient, customizable in-house AI solution.

Developed and fine-tuned "LinguaLink," an AI tool for translating technical documents (e.g., product manuals) based on the Transformer architecture. Utilized the pre-trained Helsinki-NLP/opus-mt-en-fr model and fine-tuned it specifically on internal Atlas Copco Translation Memory (TM) data. The process included data preprocessing, generating embeddings, model training over 20 epochs, and using RegX for stylesheet-compliant output formatting. LinguaLink provides faster, cost-effective, and adaptable translations compared to external vendors and outperformed baseline models like GPT-4o and T5-small in specific evaluations.

Key Technologies & Concepts: Natural Language Processing (NLP) Machine Translation Transformer Models Helsinki-NLP Opus-MT Model Fine-tuning Python BLEU Score Evaluation Regular Expressions (RegX) JSON

Results: Achieved a BLEU score of 0.7+ (outperforming benchmarks) and a Language Quality Index of 71%, meeting internal requirements while significantly reducing costs and turnaround time.
